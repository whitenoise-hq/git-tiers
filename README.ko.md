## Git TIERS란?

Git TIERS는 GitHub 활동을 기반으로 **Iron**부터 **Challenger**까지 티어를 부여합니다. 커밋, 이슈, PR, 코드 리뷰, 위키 작성이 모두 반영됩니다.

티어 뱃지를 생성하고 GitHub README에 삽입하여 기여도를 보여주세요.

## 사용 방법

| 단계 | 설명 |
|------|------|
| **1** | GitHub 계정으로 로그인 |
| **2** | 마이페이지에서 티어 확인 |
| **3** | 유형과 배경색 커스터마이즈 |
| **4** | **이미지 저장** 후 **태그 복사**하여 README에 붙여넣기 |

## 삽입 예시

```html
<a href="https://github.com/git-tiers/gittiers">
  <img src="https://git-tiers.devwoodie.com/api/tier/YOUR_GITHUB_ID" alt="Git-TIERS" />
</a>
```
